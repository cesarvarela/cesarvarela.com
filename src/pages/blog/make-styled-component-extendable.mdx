# Make your custom component easily extendable with styled-components

```javascript
{ 
	resolve: `gatsby-plugin-mdx`, 
	options: { 
		defaultLayouts: {
			default: require.resolve("./src/components/PostLayout.js"),
		},
	},
},
```
